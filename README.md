# phpMyChroma

**phpMyChroma** is a tiny PHP 8+ web client that lets you:

- Browse ChromaDB tenants ‚Üí databases ‚Üí collections ‚Üí documents
- Add / Edit / Deletion of  databases ‚Üí collections ‚Üí documents
- Semantic Search of documents with in a collection


Open http://localhost:8080 in your browser
